# BedWars Event Plugin

Расширенный движок ивентов BedWars, который выделяется среди стандартных
режимов. Плагин умеет очищать арену после матча, поддерживает "фокусы"
(динамические сценарии) и подходит для специальных турниров.

## Основные возможности
- **Авточистка карт**: после завершения боя арена восстанавливается в
  исходное состояние, что экономит время на подготовку.
- **Фокусы / Focus Events**: можно включать особые условия (например,
  приоритет на атаку конкретной команды или усиленные генераторы).
- **Таймеры и триггеры**: внутриигровые события запускаются по времени
  или админ-командам (увеличение скорости генераторов, выдача лута и т.п.).
- **Совместимость**: работает поверх стандартных арен BedWars, не
  требуя правок основного плагина.
- **API-хуки**: другие плагины могут реагировать на начало/конец
  ивента через менеджеры `BedWarsManager` и вспомогательные классы.

## Команды
| Команда | Описание | Право |
|---------|----------|-------|
| `/bedwars start <arena>` | Запуск ивента на указанной арене | bedwars.admin |
| `/bedwars stop` | Принудительное завершение текущего ивента | bedwars.admin |
| `/bedwars focus <team>` | Включить режим фокуса на команду | bedwars.admin |

> Синтаксис команд может отличаться в зависимости от настроек
> `plugins/BedwarsEvent/config.yml`. Проверьте TAB‑подсказки на сервере.

## Конфигурация
```yaml
event:
  auto-reset: true
  focus-duration: 120
  rewards:
    winner-money: 500
    winner-items:
      - DIAMOND_SWORD
```
- `auto-reset` — очищать ли арену автоматически
- `focus-duration` — длительность режима фокуса (сек)
- `rewards` — награды победителям

## Интеграция
- **WebX Panel**: отражает статус ивента, позволяет инициировать запуск
  через REST API (если включено).
- **Economy**: используется для выдачи наград монетами.
- **SharedPluginDatabase**: хранит историю победителей и статистику.
