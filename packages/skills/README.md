# Skills Plugin

Полностью настраиваемая система прокачки, позволяющая выдавать игрокам
новые способности за активную игру. Плагин поставляется с базовыми
деревьями навыков, но каждое дерево можно переписать в конфигурации без
рестартов сервера.

## Основные возможности
- **Разблокировка в процессе игры**: навыки открываются при выполнении
  условий (уровень, квесты, время в игре, статистика).
- **Геймплейные бонусы**: дополнительные эффекты к урону, защите,
  скорости, сбору ресурсов, крафту и т.д.
- **Многоуровневые деревья**: каждая ветка задаётся списком узлов с
  требованиями и цепочками зависимостей.
- **Настройка через конфиг**: файл `plugins/Skills/skills.yml`
  содержит все категории, стоимости, ограничения и иконки GUI.
- **WebX Panel мониторинг**: статистику разблокированных навыков можно
  просматривать в Dashboard и использовать для аналитики.

## Поток работы
1. Отредактируйте `skills.yml`, задав ветки, узлы, затраты очков и
   эффекты.
2. Перезапустите плагин или выполните команду перезагрузки (если
   предусмотрена админ-командой) для применения изменений.
3. Игроки зарабатывают очки навыков и тратят их в GUI, открывая
   выбранные ветки.
4. Эффекты применяются автоматически и могут быть считаны другими
   плагинами через SharedPluginDatabase.

## Конфигурация
Пример фрагмента `skills.yml`:
```yaml
skills:
  combat:
    display: "⚔ Combat"
    icon: DIAMOND_SWORD
    nodes:
      rage:
        cost: 1
        bonus: +5% damage
        requires: []
      guardian:
        cost: 2
        bonus: +2 hearts
        requires: [rage]
```
- `cost` — цена в очках навыков
- `bonus` — описание эффекта (логика прописана в коде плагина)
- `requires` — список узлов, которые нужно открыть раньше

## Интеграция
- **PlayerInfo / WebX Dashboard**: может отображать топ навыков игрока.
- **SharedPluginDatabase**: хранит текущее состояние навыков, что позволяет
  другим плагинам читать эти данные через JSON-хранилище.
